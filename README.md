# TabTamer 🗂️

A browser tab management UI that helps you fight the chaos of too many open tabs.

## Features

- **Tab Groups** — Organize tabs into color-coded groups (Work, Research, Dev, etc.)
- **Bulk Actions** — Select multiple tabs to close, pin, or move them all at once
- **Smart Filters** — Filter by Old tabs (3+ days), Recent, or Pinned
- **Search** — Instantly search across tab titles and URLs
- **Stats Dashboard** — See total, old, pinned, and grouped tab counts at a glance

## Tech Stack

- Vanilla JavaScript (no frameworks)
- HTML5 & CSS3
- Zero dependencies
